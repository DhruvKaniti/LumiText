### LumiText - AI Accessibility Helper
AI-powered accessibility for text, images, speech, and language.

## Description
LumiText is a **web-based accessibility tool**  that helps make informtion easier to understand and interact with. It combines multiple AI models to provide:

1. **Text Simplification** – Makes complex text easy to read and screen-reader friendly.

2. **Image Description** – Generates clear descriptions for images for visually impaired users.

3. **Speech-to-Text** – Transcribes audio into readable text.

4. **Translation** – Automatically detects source language and translates to any target language.
